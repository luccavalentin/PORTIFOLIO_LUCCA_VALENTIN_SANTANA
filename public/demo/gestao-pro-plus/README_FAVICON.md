# Favicon do Sistema Gestão PRO +

O favicon do sistema foi atualizado para usar o novo design com as iniciais "GP+".

## Arquivos de Favicon

- `favicon.svg` - Favicon moderno em formato SVG (suportado por navegadores modernos)
- `favicon.ico` - Favicon tradicional (se necessário, converta o SVG para ICO)

## Como converter SVG para ICO (opcional)

Se você quiser substituir o `favicon.ico` antigo, você pode:

1. **Usar um conversor online:**
   - https://convertio.co/svg-ico/
   - https://favicon.io/favicon-converter/
   - https://realfavicongenerator.net/

2. **Passos:**
   - Acesse um dos sites acima
   - Faça upload do arquivo `favicon.svg`
   - Baixe o arquivo `favicon.ico` gerado
   - Substitua o arquivo `public/favicon.ico` pelo novo arquivo

## Limpar cache do navegador

Após substituir o favicon, você pode precisar limpar o cache do navegador:

- **Chrome/Edge:** Ctrl + Shift + Delete → Limpar dados de navegação
- **Firefox:** Ctrl + Shift + Delete → Limpar dados recentes
- Ou use uma janela anônima para testar

## Verificação

O favicon deve aparecer na aba do navegador após:
1. Substituir o arquivo (se necessário)
2. Limpar o cache
3. Recarregar a página (Ctrl + F5)
